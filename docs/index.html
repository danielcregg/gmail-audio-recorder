Features

Everything you need,
nothing you don't

🎤

One-click recording

Mic button sits right next to Send in every compose window. Click, speak, attach — that's it.

📝

Live transcription

Words appear in real-time as you speak using Chrome's built-in speech recognition. No external services.

🎧

Universal playback

Records in WAV format — plays natively on every device, every email client, every OS. Zero friction for recipients.

🔒

Fully private

No data leaves your browser. Recording and transcription happen entirely on-device. No servers, no accounts.

📊

Waveform visualizer

Live animated waveform so you always know the mic is picking you up. Clean, satisfying visual feedback.

📩

Inline transcript

Optionally inserts a formatted transcript block into the email body so recipients can read without opening the attachment.

Getting started

Install in under a minute

Chrome Web Store

Click "Add to Chrome" on the Chrome Web Store listing. One click and you're done.

Or install manually

Download the source from GitHub, unzip it, then open chrome://extensions

Enable developer mode

Toggle Developer mode in the top-right corner of the extensions page.

Load the extension

Click "Load unpacked" and select the unzipped folder. Open Gmail — the mic button is ready.
